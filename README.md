# MyBlog 个人博客系统

这是一个基于 Django 开发的个人博客系统，支持用户注册、登录、发布博客、评论等功能。

## 功能特性

- **用户认证**：支持用户注册、登录和退出功能，并通过邮箱验证码进行身份验证。
- **博客管理**：用户可以发布博客，支持富文本编辑器（wangEditor）撰写内容。
- **分类管理**：博客可按分类展示，方便管理和浏览。
- **评论系统**：用户可以在博客详情页发表评论。
- **搜索功能**：支持根据标题或内容关键词搜索博客。
- **响应式设计**：前端页面采用 Bootstrap 框架，适配多种设备。

## 技术栈

### 后端
- Python 3.13
- Django 6.0.1
- MySQL 数据库

### 前端
- HTML/CSS/JavaScript
- jQuery
- Bootstrap 5
- wangEditor 富文本编辑器

### 其他工具
- 邮件服务（QQ 邮箱 SMTP）

## 安装与运行

### 环境准备
1. 安装 Python 3.13 和 pip。
2. 安装 MySQL 并创建数据库。
3. 在 `myblog/my.cnf` 中配置数据库连接信息。

## 使用说明

1. 注册账号并登录。
2. 在首页查看所有博客或通过搜索功能查找感兴趣的内容。
3. 点击“发布博客”按钮撰写新文章，支持富文本编辑。
4. 在博客详情页可以阅读文章并发表评论。
